How Binary to Decimal Works

Step-by-step Formula

  • 1. Start from the rightmost bit (LSB)
  • 2. Assign powers of 2: 2⁰, 2¹, 2², ...
  • 3. Multiply each bit by its power value
  • 4. Sum all the results

Example: 1010₂

1×2³ + 0×2² + 1×2¹ + 0×2⁰

= 8 + 0 + 2 + 0

= 10₁₀
